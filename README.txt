
простий сайт тесту
- index.html — інтерфейс
- script.js — логіка
- styles.css — стилі
- quiz.json — питання і варіанти; перша відповідь у кожному питанні є правильною; опції перемішуються

як запустити локально
- відкрий index.html у браузері
- якщо браузер блокує fetch файлів з файлової системи, підніми локальний сервер, напр. у Python:
  python -m http.server 8000
  і відкрий http://localhost:8000

як безплатно захостити
варіант 1 — GitHub Pages
- створи новий публічний репозиторій
- завантаж усі 4 файли зі цієї папки у корінь репозиторію
- у Settings → Pages обери Branch main і /root
- за 1–2 хвилини сайт буде доступний за посиланням https://<твій-юзернейм>.github.io/<назва-репозиторію>/

варіант 2 — Netlify Drop
- зайди на app.netlify.com/drop
- перетягни папку з файлами на сторінку
- Netlify розгорне статичний сайт і дасть посилання
